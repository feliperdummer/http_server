HTML, CSS e JavaScript

HTML - Hypertext Markup Language:

Não é linguagem de programação;
Focada em conteúdo (Texto, vídeo, foto, documento, etc);

CSS - Cascading Style Sheets:

Não é linguagem de programação;
Focada em design (Cores, sombra, tamanho posicionamento, etc);

JS - JavaScript:

Linguagem de programação;
Focada em interatividade (Menus, interação, pop-ups, validação);

Tags

<html></html>: raiz do documento, todo elemento deve estar dentro deste elemento;

<head></head>: Informações do documento que não são mostradas na tela, como metadados e etc;

<meta>: são tags utlizadas para inserir metadados em uma página;

<body></body>: Contém todo o conteúdo exibido na página, como textos, imagens, videos e etc;

<h1></h1>: Indicam um elemento de cabeçalho. Servem para especificar que partes do conteúdo referem-se a titulos ou subtitulos. Um documento HTMl pode ter até 6 niveis de titulo, indo de <h1> até <h6>;

<p></p>: Indicam parágrafos de texto;

<code></code>: Indicam pedaços de código de computador; OBS: tags podem conter atributos, como em lang na tag head;
